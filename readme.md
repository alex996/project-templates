## Usage

1. Clone the repo
```sh
git clone https://github.com/alex996/project-templates.git && cd project-templates
```

2. Pick a template
```sh
cd react-webpack
```

3. Install the deps
```sh
npm i
```

4. Run the dev server
```sh
npm start
```

5. Enjoy! :tada:
